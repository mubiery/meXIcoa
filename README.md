# 🐍 Tutorial Python — Fecha, Hora y Nombre

Tutorial paso a paso en GitHub Pages que explica cómo crear un programa en Python que imprime la fecha y hora actual, pide el nombre del usuario y lo guarda en un archivo.

## 📁 Archivos

| Archivo | Descripción |
|---|---|
| `index.html` | Página principal del tutorial |
| `style.css` | Estilos (diseño minimalista elegante) |
| `script.js` | Interactividad (copiar código, animaciones) |
| `README.md` | Este archivo |

## 🚀 Ver en GitHub Pages

Una vez publicado, la página estará disponible en:
```
https://tu-usuario.github.io/nombre-del-repositorio
```

## 🛠 Publicar en GitHub Pages

1. Sube estos archivos a un repositorio en GitHub
2. Ve a **Settings → Pages**
3. En "Source" selecciona la rama `main` y la carpeta `/ (root)`
4. Guarda los cambios
5. ¡Listo! En unos minutos tu página estará publicada

## 📚 Tecnologías

- HTML5 semántico
- CSS3 con variables y animaciones
- JavaScript vanilla (sin dependencias)
- Fuentes: Playfair Display + JetBrains Mono + Lato (Google Fonts)
